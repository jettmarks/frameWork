Provides the framework's main(String args[]) method as well as abstract class that should be implemented/extended to create an application's entry point. {@link com.jettmarks.base.ArgParsingMain} is the application's entry point from a java perspective. {@link com.jettmarks.base.ArgParsingEntryPoint} is the class extended by the application to provide the list of options recognized by the app and also the entry point for the first thread of execution.
